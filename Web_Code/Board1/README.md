# PHP 게시판

아래의 명령어를 통해 코드 git clone을 진행합니다.

```bash
sudo git clone https://github.com/OCSF-Logrrr/Linux.git
```

이후 Board1 디렉토리 안에 존재하는 파일들을 `var/www/html` 경로에 위치하도록 합니다.
